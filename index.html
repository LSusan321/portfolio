PROJECT · 情绪疗愈

MoodSpring 情绪疗愈

基于 CBT 的轻量情绪管理 Web 应用 · 独立开发 · 2025

项目背景

在回龙观医院实习期间完成用户需求挖掘，发现三大痛点：情绪命名混乱、记录后缺乏分析引导、疗愈环节缺少趣味性。结合 CBT「认知-情绪-行为」理论，设计「记录→分析→调节」闭环。

核心功能
  • 8维情绪模型（Plutchik情绪罗盘）+ 自定义滑条
  • DeepSeek API驱动的CBT对话，支持情绪画像注入
  • 绘画疗愈游戏 × 正念呼吸引导（1-3分钟，5种音效）
  • 情绪趋势可视化：折线图/饼图/日历热力图/30天回溯
  • 数据导入导出、配图日记、离线降级
技术实现
  • AI辅助编程（CodeBuddy）4周完成5功能模块
  • 3000+行代码，15个迭代版本完整记录
  • DeepSeek API + localStorage + Express代理
PROJECT · 互动叙事

「我只是想拿全勤」

日常荒诞题材 · 多分支视觉小说 · 灵光AI无代码平台 · 2025

项目背景

轻量互动叙事赛道集中在悬疑和养成，"日常荒诞"几乎空白。以"打工人全勤"痛点为主题，将日常场景与黑色幽默结局结合。

核心设计
  • 3大关 × 10子关卡的多分支叙事结构
  • 40+交互节点：场景点击 + 选项卡片双模式
  • 15成就稀有度分级（SSR→B），死亡回溯机制
  • 随机判定 + 隐藏彩蛋 + 4种if线结局
产品成果
  • 130+轮AI对话协作迭代
  • 解决PC/移动端兼容性 + 新手引导缺失问题
  • 潜力评分 4/5，题材击中打工人痛点
PROJECT · 自动化工作流

AI × 心理学日报

基于 WorkBuddy 的自动化 Agent Pipeline · 2025

项目背景

需要持续追踪AI与心理学交叉领域最新进展，手动检索12+期刊效率极低。利用WorkBuddy搭建自动化Pipeline，实现全流程自动化。

系统架构
  • 学术源：Nature / PNAS / JAMA / Frontiers 等 12+ 期刊
  • 采集策略：PubMed API批量筛选 + WebSearch精准补充
  • Pipeline：自动采集→去重筛选→结构化摘要→交叉引用→可视化
  • 交付：HTML日报，含效应量数据、原文引用、趋势洞察
进化轨迹
  • Alpha：5-6篇/期 → V3.0：10+篇/期，文章间交叉引用
  • 14天完成4次架构迭代，建立故障模式库
PROJECT · 竞品分析

竞品情报监控系统

基于 Agent 流水线 · 竞品分析+自动化搭建 · 2025

项目背景

MoodSpring开发前期需系统性了解竞品功能布局。设计Agent流水线竞品监控系统，覆盖 Daylio、Headspace、Moodfit 等 6 款竞品，实现「采集→对比→分析→报告」全链路自动化。

功能设计
  • 4协作Agent（采集/对比/分析/报告），全链路自动化
  • 可视化仪表盘：变更时间线 + 详情卡片 + 雷达图 + 象限分析
  • 单轮7项竞品变更 + 影响评级（高/中/低）
产品成果
  • 提炼3条关键应对策略
  • 锁定「情绪粒度记录 + CBT对话 + 正念呼吸 + 艺术疗愈」核心功能
  • 建立自动化执行日志与可维护知识库
